## After clone, do

1. Install dependencies

   ```sh
   yarn install
   ```

2. Create `.env` file from `.env.template`. Then, Fill in the required environment variables in the `.env` file.

   ```sh
   cp .env.example .env
   ```

3. Get up Docker environment

   ```sh
   docker-compose up -d
   ```

4. Execute app

   ```sh
   yarn start:dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)
